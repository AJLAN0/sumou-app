🏢 نظرة عامةOverview

تطبيق سمو الإبداع هو نظام إدارة داخلي متكامل لشركة تصوير إنتاج مرئي. يشمل إدارة المشاريع، الزواجات، الحضور، الصلاحيات، والتسليمات — مع ربط خارجي بمنصة ركاز للحجوزات.
Sumou Creative is a comprehensive internal management system for a visual production company. It covers project management, weddings, attendance, permissions, and deliverables — with external integration via Rekaz booking platform.

الهدف الرئيسيMain Goal

بوابة موحدة يدخل منها كل موظف (مدير، مصور، مصمم، إداري) لإدارة مهامه — بينما يتابع العميل مشروعه من خارج التطبيق عبر رمز سري.
A unified portal for all staff (manager, photographer, designer, admin) to manage their tasks — while clients track their projects externally via a secret code.

٣ / 3 — نظام القدرة الاستيعابيةCapacity System

كل مصور له حد شهري. المصور الممتلئ يظهر بالأحمر ومُعطَّل في قائمة الاختيار. الحساب شهري وليس يومياً.
Each photographer has a monthly capacity limit. A full photographer appears in red and is disabled in the selection list. Calculation is monthly, not daily.

٤ / 4 — نظام التقييم والستريكRatings & Streak

٥ / 5 — نظام الصلاحيات الديناميكيDynamic Permissions

كل مستخدم يحمل أدواراً متعددة. المصور تُحدَّد له أنواع التصوير ويظهر في القوائم المناسبة. تعطيل الحساب يمنع الدخول نهائياً.
Each user can hold multiple roles. Photographer types define which project lists they appear in. Disabling an account blocks all access.

٦ / 6 — ربط ركازRekaz Integration

حجوزات ركاز تصل عبر Webhook إلى Supabase وتظهر في التطبيق. حجوزات الزواجات تظهر كـ"طلب جديد" لإدارة الزواجات لتحويلها لمشروع.
Rekaz bookings arrive via Webhook → saved to Supabase → displayed in app. Wedding bookings appear as "New Request" for the wedding admin to convert to a project.
